## DIGITAL-CLOCK ⏰

A simple digital clock built with **REACT** . This project display the current time and update every second.

---

## Features

1. Showa hours, minutes, and seconds
2. updates automatically in real-time
3. Clean and minimal design


---

## Technology

- **React**
- **JavaScript** (ES6+)
- **Vite** (development server & build tool)
- **CSS** (for styling)
